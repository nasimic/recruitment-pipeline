# Installation guide

- Clone project

- Install packages
```
composer install
```

- Copy and paste **.env.example** (Rename it to **.env**)
```
cp .env.example .env
```

- Change DB configs in **.env** file as you wish
```
php artisan optimize
```
```
php artisan key:generate
```
```
php artisan migrate
```
```
php artisan db:seed
```

**Enjoy**
